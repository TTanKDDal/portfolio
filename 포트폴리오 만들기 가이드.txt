포트폴리오(= 개인 웹사이트)
: 이력서, 자소서와 함께 디자인 감각을 드러낼 수 있는 표현 수단

깃허브: HTML, CSS, Javascript등 코드를 저장할 수 있는 저장소

깃허브 사용방법
1. 깃허브 사이트 로그인
2. '깃허브 데스크톱' 다운로드
- [File - clone respository..] URl 탭 후 URL 복붙
- Open in Visual Studio Code 버튼 클릭 -> VSCode 프로그램 연결
- Commit to~ 버튼 클릭(Commit: 변경사항 이력)
- Push 클릭하여 파일 업로드
- Push = 업로드, Pull = 다운로드

1. 비주얼
- '나'라는 사람이 어떤 사람인지
- 자신의 작품 소개 스토리를 이어갈 첫 머리와 같은 역할을 하기 때운에 자신의 개성이 
강력한 문구, 조화로운 색채로 표현하는 것이 중요
- 슬로건 문구 또는 이미지
- 한 두줄 정도의 간단한 자기소개, 포부, 가치관 등

2. 소개
- 자기소개(사진, 이름, 학력, 경력 등)
- 디자이너로서의 정체성, 방향성 강조
- 자격증, 수상이력
- 사용 가능한 툴 및 기술 스택(아이콘 형식 추천)
-이력서, 자소서 다운받을 수 있는 버튼

3. 프로젝트 작업물(과제 + 개인작업)
- 기본과제: 웹사이트 2개, UI/UX, 앱1개, 상세페이지 1개, 프로모션페이지 1개, 브랜딩 1개
- 썸네일 + 프로젝트 제목 + 기여도(개인 100%) + 연결링크 등으로 구성

[각 프로젝트 상세페이지]
- 프로젝트 제목, 작업기간(약간 보정을 해도 무방 한달=20일/ 시간으로 표현되 괜찮다) 등 
- 프로젝트 개요: 문제정의 및 프로젝트 목표
- 디자인 콘셉트 및 간략한 스타일 가이드(색상, 서체, 디테일 아이콘 등)
- 최종 디자인(결과물) + 목업 
- 웹사이트의 경우라면 구현사이트로 바로갈 수 있는 링크
